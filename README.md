# Persona-V-Saran
Persona 5–inspired interactive portfolio featuring a kinetic UI/UX system with animated case files, IAM-focused experience, and full-stack + AI/ML projects. Built using React, Tailwind CSS, and Framer Motion.

# 🃏 Persona Portfolio — SARAN SHABU

> An aggressive, high-energy personal portfolio inspired by the UI/UX of Persona 5.

---

## ⚡ Overview

This is not a traditional portfolio.

It is a **fully interactive, stylized experience** that blends:
- Cybersecurity & IAM expertise
- Full-stack development
- AI/ML research work

Designed with a **kinetic, chaotic, and cinematic UI**, the goal is to present projects and experience like **classified dossiers and system interfaces**.

## 🧠 Core Features

### 🔴 Hero System
- Slanted typography and layered geometry
- Anime-style character integration
- Dynamic visual hierarchy

### 📁 Case Files (Projects)
- Featured projects with immersive hover states
- Background image reveals per project
- Classified archive system for additional projects

### 🧬 About Section (Interactive)
- Expandable stat cards
- Education timeline (animated)
- Research publication dossiers

### 🧰 Arsenal (Skills)
- Stylized tech clusters
- Persona-inspired tag system

### 🧾 Operations Log (Experience)
- Timeline-driven layout
- IAM + enterprise exposure highlighted

### 🌐 Network Footer
- Compact multi-link system (GitHub, LinkedIn, TryHackMe, etc.)

---

## 🎨 Design Philosophy

- ❌ No rounded components  
- 🔺 Heavy use of `clip-path` geometry  
- ⚡ Snappy animations (no slow fades)  
- 🟥 Strict color palette:
  - Red (#FF0000)
  - Charcoal Black (#1A1A1A)
  - White (#FFFFFF)

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Design System:** Custom Persona-inspired UI logic

## ⚙️ Installation & Setup

```bash
# Clone the repo
git clone https://github.com/yourusername/persona-portfolio.git

# Navigate into project
cd persona-portfolio

# Install dependencies
npm install

# Run locally
npm run dev
```

